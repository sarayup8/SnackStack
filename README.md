# SnackStack

SnackStack is a simple recipe discovery app that helps users quickly find meals. Users can shuffle through random recipes or search for specific dishes, then view detailed instructions and ingredients.

---

## API + Tools Used

- **API:** TheMealDB  
- **Language:** Swift  
- **Framework:** SwiftUI  
- **Development:** Xcode

## Features

- **Random Recipe Generator** - Tap “Shuffle” to get a new random meal  

- **Search Functionality** - Search for recipes by name + sort by category

- **Detailed Recipe View** - View ingredients, measurements, and instructions  

- **Image Loading**  - Uses `AsyncImage` to display recipe thumbnails  

- **Live API Integration** - Fetches real-time data from TheMealDB   
---
